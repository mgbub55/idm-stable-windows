IDM controlla ed intercetta una richiesta di download da qualsiasi browser, se il file corrisponde ad un tipo elencato nella lista dei tipi di file. Puoi modificare questa lista nel pannello "Opzioni->Tipi di file".
IDM intercetterà qualsiasi download se tu terrai premuto uno dei tasti speciali, mentre clicchi sul link del file da scaricare (dovrai prima impostare questi tasti, cliccando sul pulsante "Tasti", nel pannello "Opzioni->Generale". Sappi che il tasto "Ins" della tua tastiera dovrebbe andar bene in quasi tutti i browser).
Se tieni premuto il tasto ALT mentre clicchi su un link da scaricare, IDM non intercetterà quel download e lascerà che sia il browser a scaricarlo (puoi cambiare questo tasto, cliccando sul pulsante "Tasti", nel pannello "Opzioni->Generale").
Se hai bisogno di conoscere meglio qualche funzionalità di IDM, premi il tasto F1 e si aprirà la Guida di IDM.
Se non desideri che IDM controlli i download del browser, basterà che disattivi l'integrazione browser nel pannello Opzioni.
Puoi trascinare un link da un browser e rilasciarlo nel "Box di rilascio". Questa finestrella apparirà se cliccherai col pulsante destro del mouse sull'icona di IDM nell'Area di Notifica (oppure nel menu "Operazioni" di IDM), e attiverai la voce "Mostra 'Box di rilascio".
Puoi disabilitare la finestra di dialogo "Download completato", che chiede se aprire il file. Basta che deselezioni l'opzione "Mostra finestra 'Download completato" nel pannello "Opzioni->Download".
Puoi disabilitare la finestra di dialogo "Info download", che chiede dove salvare il file. Basta che deselezioni l'opzione 'Mostra finestra 'Info download' nel pannello "Opzioni->Download".
Puoi ridurre ad icona o disabilitare la finestra di progressione dei download, nel pannello "Opzioni->Download".
Puoi eseguire IDM anche da linea di comando e scaricare file. Per informazioni più dettagliate, consulta il capitolo "Linea di comando" nella Guida di Internet Download Manager.
Puoi trascinare i file dalla finestra principale di IDM e rilasciarli in altre applicazioni (per esempio, Winamp, Media Player, ecc.). Puoi anche usare il "drag & drop" per spostare o copiare i file da IDM a qualsiasi altra cartella di Windows.
Se non vuoi immettere continuamente, ogni volta che scarichi da un server protetto, lo stesso nome utente e password, puoi aggiungere il server con relativi nome utente e password, nel pannello "Opzioni->Siti con login".
Se hai problemi nel download dei file da uno specifico server (ad esempio server occupato, continui errori di timeout ed altri problemi inerenti ai server e alle reti), puoi lasciare questo file in una coda di sincronizzazione, tramite la funzionalità "Pianifica", e poi avviarla. IDM proverà a scaricare il file, facendo un tentativo ogni 30 secondi.
