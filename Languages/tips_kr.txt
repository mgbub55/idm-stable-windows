"IDM 환경설정->일반->다음 파일 확장자 감지시 자동으로 내려받기 시작" 에 등록된 파일 확장자 감지시 어떤 브라우저에서라도 IDM는 내려받기 작업을 시도합니다.  파일 확장자는 사용자가 직접 추가, 삭제, 관리할 수 있습니다.
브라우저에서 내려받을 링크를 클릭할 때 CTRL 키를 같이 누를 경우 IDM에서는 모든 내려받기 작업을 수행하게 됩니다. ("IDM 환경설정->일반->IDM으로 내려받기 위해 단축키 지정"에서 해당 단축키를 설정할 수 있습니다)
브라우저에서 내려받을 링크를 클릭할 때 ALT 키를 같이 누를 경우, IDM에서는 해당 링크의 파일을 내려받지 않습니다. ("IDM 환경설정->일반->IDM으로 내려받기 위해 단축키 지정"에서 해당 단축키를 설정할 수 있습니다)
궁금하신게 있으실경우, F1 키를 누르면 자세한 도움말을 확인할 수 있습니다.
브라우저에서 IDM 프로그램으로 내려받기 작업을 사용하지 않으려면, IDM 환경설정에서 브라우저 연결 기능 사용을 해제하십시오.
웹 브라우저의 링크를 끌어서 놓기 기능을 사용하여 "드롭존" 상자에 떨어뜨릴 수 있습니다. "드롭존" 상자는 IDM 시스템 트레이 아이콘 상에서 마우스 오른쪽 버튼을 클릭하여 나오는 메뉴에서 "드롭존 보기"를 선택하여 사용할 수 있습니다. 또는, IDM 메인 메뉴의 "주소->드롭존 보기"를 선택할 수도 있습니다.
내려받기 완료 대화 창을 표시하지 않을 수 있습니다. "IDM 환경설정->내려받기" 탭에서 "내려받기 완료 대화 창 보기" 항목을 선택 해제하십시오.
내려받기 시작 대화 창을 표시하지 않을 수 있습니다. "IDM 환경설정->내려받기" 탭에서 "내려받기 시작 창 보기" 항목을 선택 해제하십시오.
내려받기 진행 상황 창의 상태를 "IDM 환경설정->내려받기"에서 선택할 수 있습니다.
IDM에서 파일을 내려받기 위하여 명령어를 사용할 수 있습니다. 자세한 내용은 "도움말->내려받기 시작" 챕터에서 확인할 수 있습니다.
사용자는 IDM 메인 화면에서 파일을 끌어서 다른 프로그램(예시, Winamp, Media Player, 기타.)으로 떨어뜨릴 수 있습니다. 또한 IDM에서 끌어서 놓기 기능을 사용하여 탐색기 폴더로 파일을 복사 또는 이동할 수 있습니다.
인증을 요구하는 서버에 사용자의 이름과 비밀번호 입력을 매번 수행하지 않으려면, "IDM 환경설정->인증관리" 탭에서 로그인 이름과 비밀번호를 요구하는 사이트를 추가하여 관리할 수 있습니다.
서버로부터 파일을 내려받는데 어려움이 있을 경우 (예시, 서버 분주, 무한 대기시간, 다른 서버와 연관된 네트워크 오류 등), IDM 예약 작업에서 해당 파일을 예약하여, 매 30초 간격으로 내려받기를 자동으로 시도합니다.
