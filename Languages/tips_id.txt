IDM mengambil alih download dari semua browser jika jenis filenya cocok dengan jenis/ekstensi file yang ada di daftar. Anda dapat mengedit daftar ekstensi di "Opsi->Jenis File".
IDM akan mengambil alih semua download jika Anda menekan dan menahan tombol khusus sambil mengklik link download di browser. (Anda perlu mengatur tombol ini dalam Opsi IDM, Tab Umum, Tombol. Perhatikan bahwa tombol "Insert" seharusnya bekerja hampir di semua browser)
Jika Anda tekan-tahan tombol ALT sambil mengklik link download di browser, IDM tidak akan mengambil alih download dan membiarkan file didownload oleh browser. (Anda dapat mengubah tombol ini dalam Opsi IDM, Tab Umum, Tombol)
Jika Anda tidak mengerti apa-apa, coba saja tekan tombol F1 dan bantuan terkait yang akurat akan muncul.
Jika Anda tidak ingin IDM mengambil alih setiap download dari browser, matikan integrasi browser dalam Opsi IDM.
Anda dapat menyeret link dari browser web dan meletakkannya di keranjang "Target yang didrop". Keranjang "Target yang didrop " akan muncul ketika Anda mengklik kanan ikon tray IDM dan memilih "Tampilkan target yang didrop " atau ketika Anda memilih "Url->Tampilkan target yang didrop " dari jendela utama IDM.
Anda dapat mematikan dialog selesai download yang menyarankan untuk membuka file. Hapus saja centang di kotak "Tampilkan dialog selesai download " pada "Opsi IDM->Download"
Anda dapat mematikan dialog memulai download yang menanyakan di mana akan menyimpan file. Hapus saja centang di kotak "Tampilkan dialog memulai download " pada "Opsi IDM->Download"
Anda bisa meminimalkan atau menonaktifkan dialog progress download di Opsi IDM -> Download.
Anda dapat menjalankan IDM dari baris perintah untuk mendownload file. Baca bab "Bantuan-> Memulai Download" untuk informasi lebih lanjut.
Anda dapat menyeret file dari jendela utama IDM dan meletakkannya ke aplikasi lain (misalnya, Winamp, Media Player, dll). Anda juga dapat menggunakan drag and drop untuk memindahkan atau menyalin file dari IDM ke folder manapun di Windows.
Jika Anda tidak ingin memasukkan nama dan password yang sama berulang-ulang setiap kali Anda mendownload dari server yang terproteksi, Anda dapat menambahkan server, nama login dan password ke daftar login situs di "Opsi IDM->Login Situs"
Jika Anda memiliki masalah dalam mendownload sebuah file dari server (seperti server sibuk, sering timeout, dan masalah lainnya terkait server atau jaringan), Anda dapat meninggalkan file tersebut sendirian di pengatur jadwal antrian IDM dan memulai antriannya. Pengatur jadwal IDM akan coba mendownload file setiap 30 detik
